# Plan 5.4: Gallery Page Review & Fixes

Review and fix Gallery page to improve image browsing, organization, and management functionality.

## Overview

The Gallery page provides image browsing and organization capabilities. This plan addresses improvements to view modes, organization, performance, and management features.

## Tasks

### 1. View Modes
- Grid view: Responsive masonry or uniform grid
- List view: Table with sortable columns (date, name, camera, size)
- Timeline view: Group by date with collapsible sections
- Add toggle buttons in toolbar
- Persist view preference in localStorage

### 2. Organization System
- Create albums/folders for organizing images
- Drag-and-drop images to albums
- Multi-select with Shift+Click and Ctrl+Click
- Album sidebar with quick access
- Create album from selection
- Album sharing capability

### 3. Performance Optimization
- Implement virtual scrolling for large galleries
- Lazy load images with intersection observer
- Use thumbnail images for grid, full size on demand
- Add loading skeletons
- Implement image prefetching on hover
- Cache frequently accessed images

### 4. Bulk Operations
- Select all / Deselect all
- Bulk move to album
- Bulk delete with confirmation
- Bulk export (ZIP download)
- Bulk add tags/labels
- Operation progress indicator

### 5. Favorites & Bookmarks
- Star/favorite images
- Dedicated "Favorites" album
- Filter by favorites
- Keyboard shortcut to toggle favorite

### 6. Image Detail View
- Full-size image viewer with zoom
- Pan functionality for zoomed images
- Image metadata panel (EXIF data, detection info)
- Detection annotations toggle
- Navigation: Previous/Next buttons
- Download original button
- Share image link

### 7. Search & Filters
- Search by filename, date, camera, tags
- Date range filter
- Camera filter dropdown
- File type filter (all, images only, with detections)
- Sort options (date, name, size)
