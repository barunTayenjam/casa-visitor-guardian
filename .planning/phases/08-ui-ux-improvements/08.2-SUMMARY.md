## User Experience

**Before:** Users had no visibility into stream quality issues
**After:** Users can diagnose problems at a glance and get actionable suggestions

**Quality Levels:**
- **Good:** FPS ≥20, bandwidth ≥1 Mbps, latency ≤100ms
- **Warning:** FPS 10-20, bandwidth 500-1000 Kbps, latency 100-500ms
- **Critical:** FPS <10, bandwidth <500 Kbps, latency >500ms
