<objective>
Binary frame delivery over Socket.io replacing base64 encoding

Purpose: Eliminate the 33% base64 bandwidth overhead documented in ADR-002 by sending JPEG frames as raw binary over Socket.io. Frontend renders via Blob URLs with memory cleanup.

Output: Server sends Buffer directly (no toString('base64')), frontend receives ArrayBuffer and creates Blob URLs, with base64 fallback for long-polling clients.
</objective>
