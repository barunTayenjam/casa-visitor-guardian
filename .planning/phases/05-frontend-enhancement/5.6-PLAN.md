# Plan 5.6: Dashboard Review & Fixes

Review and fix Dashboard page to improve camera grid, real-time updates, and system overview functionality.

## Overview

The Dashboard page provides the main camera view and system status. This plan addresses improvements to layout, responsiveness, real-time handling, and user experience.

## Tasks

### 1. Layout & Organization
- Responsive grid layout (1-4 columns based on screen size)
- Configurable grid size (small, medium, large)
- Full-screen single camera mode
- Picture-in-picture mode for selected camera
- Save layout preference in localStorage

### 2. Camera Grid Improvements
- Drag-and-drop camera reordering
- Camera favorite/star for priority display
- Quick camera selection dropdown
- Camera health indicator overlay
- Double-click for fullscreen
- Keyboard shortcut for quick switch

### 3. Real-Time Stream Handling
- Auto-reconnect on connection loss
- Show connection status indicator
- Buffer indicator during buffering
- Adaptive quality based on network
- Stream health monitoring
- Graceful degradation when offline

### 4. System Overview Panel
- CPU/Memory usage indicator
- Active cameras count
- Events today count
- Recent alerts list
- System health status
- Collapsible for more viewing space

### 5. Quick Actions Toolbar
- Start/stop all streams
- Take snapshot (all cameras)
- Open Day Highlights
- Open Batch Detection
- Open Settings
- Refresh all feeds

### 6. Performance Optimization
- Lazy load off-screen streams
- Pause streams in background tabs
- Reduce quality when tab not visible
- Efficient frame rendering
- Memory usage monitoring
- Auto-cleanup disconnected streams

### 7. Camera-Specific Features
- Click camera to see mini stats
- Quick mute/unmute audio
- Camera info popover
- PTZ controls (if supported)
- Snapshot to clipboard
- Share camera view
