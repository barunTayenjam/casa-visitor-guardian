## Design Decisions

- All components use `cn()` from `@/lib/utils` for class merging (clsx + twMerge)
- All components use CSS variables (`bg-background`, `text-foreground`, `border-primary`, `bg-muted`, `text-muted-foreground`) — no hardcoded colors
- All components follow shadcn/ui patterns: `React.forwardRef`, `displayName`, `cn()` for className merging
- Container width: pages should use `max-w-7xl mx-auto px-4 md:px-6 py-6` (not baked into components, giving flexibility)
