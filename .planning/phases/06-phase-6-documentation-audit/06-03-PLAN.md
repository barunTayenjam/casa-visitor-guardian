**Current detection pipeline (what should be documented):**
1. Python `FFmpegReader` captures frames from RTSP via FFmpeg subprocess (raw BGR24 pipe)
2. Python `MotionGate` (MOG2 background subtraction) filters motion frames inline
3. On motion → Python `InProcessYOLO` runs YOLOv8n/YOLOv5n/YOLOv4-tiny object detection
4. Python `ByteTracker` performs temporal multi-object tracking
5. Python `IdentityEnrichment` does face recognition on new tracks (ArcFace/InsightFace)
6. Python `WebSocketPublisher` sends JPEG frames + tracking events to Node.js via WebSocket
7. Node.js `PythonWsClient` receives frames/events, `rtspManager.wirePythonWsFrames()` relays to Socket.io
8. Node.js persists tracking events as `events` in PostgreSQL with image captures
9. `consolidatedDetectionService.ts` (stub) provides type definitions only — actual detection runs in Python
