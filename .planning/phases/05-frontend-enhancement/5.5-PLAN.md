# Plan 5.5: Settings Page Review & Fixes

Review and fix Settings page to improve configuration UI, validation, and overall user experience.

## Overview

The Settings page provides system configuration options. This plan addresses improvements to organization, validation, preview functionality, and user experience.

## Tasks

### 1. Section-Based Navigation
- Add sticky sidebar with section links
- Sections: General, Cameras, Detection, Notifications, Storage, Users, About
- Smooth scroll to section on click
- Highlight current section on scroll
- Collapse/expand section support

### 2. Form Validation
- Real-time validation as user types
- Clear inline error messages below fields
- Success indicators for valid fields
- Prevent form submission with errors
- Show validation summary at top
- Validate dependent fields

### 3. Unsaved Changes
- Track form dirty state
- Show warning when navigating away
- Confirm dialog on browser close
- "Save Changes" button with loading state
- "Discard Changes" option
- Auto-save draft to localStorage

### 4. Camera Configuration
- Card-based camera list
- Inline editing of camera name
- RTSP URL with validation
- Resolution/framerate dropdowns
- Zone configuration with visual editor
- Test connection button
- Camera health status indicator

### 5. Detection Settings
- Sensitivity slider with visual indicator
- Night mode toggle with schedule
- Object type checkboxes
- Minimum confidence threshold
- Detection interval settings
- Zone-based detection toggle
- Preview of detection area

### 6. Notification Settings
- Enable/disable toggle per type
- Quiet hours configuration
- Email notification toggle
- Push notification setup
- Notification frequency limit
- Test notification button

### 7. Storage Settings
- Retention period per data type
- Auto-cleanup toggle and schedule
- Storage quota display
- Manual cleanup triggers
- Storage breakdown by type
- Warning threshold settings

### 8. User Management
- User list with roles
- Add/edit user modal
- Role assignment dropdown
- Delete user with confirmation
- Session management
- Password reset option
