# Plan 5.2: Day Highlights Review & Fixes

Review and fix Day Highlights page to improve playback controls, timeline navigation, and overall user experience.

## Overview

The Day Highlights page provides an automated slideshow of security events throughout the day. This plan addresses improvements to navigation, playback controls, export functionality, and performance.

## Tasks

### 1. Keyboard Shortcuts
- Arrow Left/Right: Previous/Next event
- Space: Play/Pause
- Home/End: First/Last event
- Number keys 1-9: Jump to 10%, 20%... 90%
- F: Toggle fullscreen
- E: Export current highlight
- Escape: Exit fullscreen

### 2. Timeline Scrubbing
- Add timeline slider below video player
- Show preview thumbnails while scrubbing
- Display event markers on timeline (color-coded by category)
- Allow click-to-seek on timeline
- Show time tooltip while hovering

### 3. Export Functionality
- Add "Export" button with dropdown menu
- Export current image with detection overlays
- Export day summary as PDF report
- Share link generation for specific highlight
- Download all images as ZIP (with selected events)

### 4. Performance Optimization
- Implement virtual scrolling for timeline list
- Lazy load images with blur-up placeholder
- Preload next 5 images during playback
- Use WebP images where supported
- Limit concurrent image loads

### 5. Mobile/Touch Improvements
- Add swipe gestures for next/previous
- Larger touch targets for controls
- Improved responsive layout for portrait mode
- Pull-to-refresh on mobile
- Pinch-to-zoom on images

### 6. Category Filtering
- Add filter chips for event categories
- Filter by: All, Persons, Faces, Known, Unknown, Night
- Quick toggle buttons in header
- Persist filter preference in localStorage

### 7. Error Handling & Edge Cases
- Handle image load failures gracefully
- Show placeholder for missing images
- Handle empty day gracefully with friendly message
- Offline mode: show cached data indicator
