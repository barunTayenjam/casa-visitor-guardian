# Plan 2.3: Event Search API

Add event search API — date range, camera, type, confidence filters for powerful event filtering.

## Overview

Create an enhanced event search API endpoint that supports multiple filter criteria, allowing users to query events by:
- Date range (start and end timestamps)
- Camera ID (cam1, cam2)
- Event type (motion, face, object)
- Detection confidence level (low, medium, high)
- Face recognition status (recognized, unknown, no face)

This requires optimized database queries, proper indexing, and flexible API design.

## Tasks

### 1. Database Indexing Strategy
- Create migration `011_event_search_indexes.sql`
- Add composite indexes for common filter combinations:
  - (timestamp DESC, camera_id) - for date range + camera
  - (event_type, timestamp DESC) - for type + date
  - (confidence, timestamp DESC) - for confidence + date
  - (faces_detected, known_faces_count, timestamp DESC) - for face status
- Add individual indexes if needed:
  - camera_id (if not exists)
  - event_type (if not exists)
  - confidence (if not exists)
- Analyze query patterns and adjust indexes accordingly

### 2. TypeORM Query Builder
- Create `server/src/services/eventSearchService.ts`
- Implement `searchEvents(filters)` function
- Use TypeORM QueryBuilder for dynamic conditions
- Filter parameters:
  ```typescript
  interface EventSearchFilters {
    startDate?: string;      // ISO timestamp
    endDate?: string;        // ISO timestamp
    cameraId?: string;       // 'cam1' | 'cam2'
    eventType?: string;      // 'motion' | 'face' | 'object'
    confidence?: string;     // 'low' | 'medium' | 'high'
    faceStatus?: string;     // 'recognized' | 'unknown' | 'none'
    page?: number;           // Default 1
    pageSize?: number;       // Default 20, max 100
    sortBy?: string;         // 'timestamp' | 'confidence' | 'camera'
    sortOrder?: 'ASC' | 'DESC'; // Default 'DESC'
  }
  ```
- Build query dynamically:
  ```typescript
  const query = eventRepository.createQueryBuilder('event');

  if (filters.startDate) {
    query.andWhere('event.timestamp >= :startDate', { startDate: filters.startDate });
  }
  if (filters.endDate) {
    query.andWhere('event.timestamp <= :endDate', { endDate: filters.endDate });
  }
  if (filters.cameraId) {
    query.andWhere('event.camera_id = :cameraId', { cameraId: filters.cameraId });
  }
  if (filters.eventType) {
    query.andWhere('event.event_type = :eventType', { eventType: filters.eventType });
  }
  if (filters.confidence) {
    const minConfidence = getConfidenceThreshold(filters.confidence);
    query.andWhere('event.confidence >= :minConfidence', { minConfidence });
  }
  if (filters.faceStatus === 'recognized') {
    query.andWhere('event.known_faces_count > 0');
  } else if (filters.faceStatus === 'unknown') {
    query.andWhere('event.faces_detected > 0')
         .andWhere('event.known_faces_count = 0');
  } else if (filters.faceStatus === 'none') {
    query.andWhere('event.faces_detected = 0');
  }
  ```
