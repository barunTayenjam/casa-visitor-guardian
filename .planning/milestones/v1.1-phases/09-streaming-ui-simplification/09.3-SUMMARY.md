- Pre-offset + rAF + animate-to-zero pattern creates directional slide (left = old content exits left, new enters from right)
- switchCameraId decoupled from focusedCameraId so animation can complete before parent state updates
- Document-level mousemove/mouseup ensures drag continues even when cursor leaves container
- Removed unused transitionDirection state (cleanup during implementation)
