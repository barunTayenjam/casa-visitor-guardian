<objective>
Replace the unsafe `(global as any)` service singleton pattern with a typed ServiceRegistry class that provides type-safe access to all backend services.

Purpose: The current `(global as any)` pattern has no type safety, makes testing impossible, creates circular dependency risks, and allows race conditions. A typed registry eliminates all these issues while maintaining the singleton access pattern.
Output: New ServiceRegistry class, migrated initialization in index.ts, updated consumers in routes and detection modules.
</objective>

    class ServiceRegistry {
      private services: Map<string, unknown> = new Map();

      setStreamManager(manager: StreamManager): void { this.services.set('streamManager', manager); }

      private getRequired<T>(key: string): T {
        const service = this.services.get(key);
        if (!service) throw new Error(`Service '${key}' not initialized`);
        return service as T;
      }
    }
