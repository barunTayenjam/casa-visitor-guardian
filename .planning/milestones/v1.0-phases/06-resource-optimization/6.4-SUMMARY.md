## Changes Made

### 1. Docker Resource Limits (docker-compose.yml)

Added resource limits to all containers using `deploy.resources`:

**Backend:**
```yaml
deploy:
  resources:
    limits:
      cpus: 1.0
      memory: 1024M
    reservations:
      cpus: 0.5
      memory: 512M
```

**OpenCV:**
```yaml
deploy:
  resources:
    limits:
      cpus: 0.5
      memory: 512M
    reservations:
      cpus: 0.25
      memory: 256M
```

**PostgreSQL:**
```yaml
command: >
  postgres
  -c shared_buffers=64MB
  -c max_connections=20
  -c work_mem=4MB
  -c maintenance_work_mem=32MB
  -c effective_cache_size=128MB
  -c huge_pages=off
deploy:
  resources:
    limits:
      cpus: 0.25
      memory: 384M
    reservations:
      cpus: 0.1
      memory: 192M
```

**Redis:**
```yaml

  redis-server
  --maxmemory 50mb
  --maxmemory-policy allkeys-lru
deploy:
  resources:
    limits:
      cpus: 0.1
      memory: 64M
    reservations:
      cpus: 0.05
      memory: 32M
```

### 2. Environment Variables for Configuration

```bash

BACKEND_MEM_RESERVE=512M

# OpenCV

# Redis

```
