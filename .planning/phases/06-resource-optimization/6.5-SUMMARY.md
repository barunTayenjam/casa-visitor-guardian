## Changes Made

### 1. LOW_RESOURCE_MODE Configuration (server/src/config/index.ts)

Configuration options are now available:
```typescript
export const config: AppConfig = {
  // ... existing config
  streaming: {
    frameInterval: number;
    lowResourceMode: boolean;
    inactivityTimeout: number;
    threads: number;
    defaultFps: number;
    defaultResolution: string;
  }
};
```

### 2. Environment Variables

**Low-Resource Mode Settings:**
```bash
# Enable low-resource mode
LOW_RESOURCE_MODE=true

# FFmpeg optimization
FFMPEG_THREADS=2
DEFAULT_FPS=2
DEFAULT_RESOLUTION=640x360
STREAM_INACTIVITY_TIMEOUT=300000

# Container limits
BACKEND_CPU_LIMIT=1.0
BACKEND_MEM_LIMIT=1024M
OPENCV_CPU_LIMIT=0.5
OPENCV_MEM_LIMIT=512M
```

### 3. Low-Resource Preset Values

For systems with 1 CPU core, 2GB RAM:
```bash
# Ultra-low resource settings
LOW_RESOURCE_MODE=true
FFMPEG_THREADS=1
DEFAULT_FPS=1
DEFAULT_RESOLUTION=320x240
STREAM_INACTIVITY_TIMEOUT=60000
NODE_MAX_MEMORY=256

```
